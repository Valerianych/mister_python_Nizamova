# Лабораторная работа 1

## Тема

Первый тест на `unittest` и базовая структура проекта.

## Цель работы

Научиться создавать простой Python-проект, отделять основной код от тестов и запускать первый автотест.

## Задание

1. Создать папку для основного кода.
2. Создать папку для тестов.
3. Добавить файл с простыми математическими функциями.
4. Написать тесты для проверки сложения, вычитания и умножения.
5. Запустить тесты через команду `python -m unittest`.

## Что должно быть в результате

В проекте должны быть:

- файл с функциями;
- файл с тестами;
- успешный запуск тестов без ошибок.

## Пример запуска

```bash
python -m unittest discover -s tests -v
```

## Вывод

В ходе работы была создана базовая структура проекта и написаны первые тесты с использованием модуля `unittest`.
