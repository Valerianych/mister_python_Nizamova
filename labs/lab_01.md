# Лабораторная работа 1

## Тема

Скелет проекта и первый тест через `unittest`.

## Цель работы

Создать минимальный проект со структурой `src` и `tests`, написать функцию `slugify` и проверить её через автотесты.

## Задание

1. Создать пакет `qautils` в папке `src`.
2. Добавить файл `src/qautils/slugify.py`.
3. Реализовать функцию `slugify(text)`.
4. Создать тестовый файл `tests/test_slugify.py`.
5. Проверить несколько вариантов входных строк.
6. Запустить тесты через discovery.

## Требования к `slugify`

Функция должна:

- переводить строку в нижний регистр;
- заменять пробелы и символ `_` на дефис;
- удалять лишние символы;
- схлопывать несколько дефисов подряд в один;
- убирать дефисы в начале и конце строки;
- возвращать пустую строку, если после очистки ничего не осталось.

## Примеры

`Hello, World!` должно стать `hello-world`.

`Already_Slug` должно стать `already-slug`.

`!!!` должно стать пустой строкой.

## Команда запуска

```bash
python -m unittest discover -s tests -t . -v
```

## Вывод

В работе создана базовая структура проекта и написаны первые тесты для функции `slugify`.
