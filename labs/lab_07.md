# Лабораторная работа 7

## Тема

`Mock` и `MagicMock`.

## Цель работы

Научиться заменять зависимость тестовой заглушкой и проверять вызовы методов.

## Задание

1. Создать сервис отправки уведомлений.
2. Передавать в него объект клиента.
3. В тестах заменить клиента через `Mock`.
4. Проверить, что метод клиента был вызван с нужными аргументами.
5. Проверить поведение при ошибке клиента.

## Команда запуска

```bash
python -m unittest tests.test_lab_07_mock -v
```

## Вывод

В работе показано использование мок-объектов для проверки взаимодействия между объектами.
