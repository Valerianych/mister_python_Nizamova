# Лабораторная работа 4

## Тема

Fixtures и жизненный цикл теста: `setUp`, `tearDown`, `addCleanup`.

## Цель работы

Показать подготовку данных перед тестом и очистку ресурсов после теста.

## Задание

1. Создать модуль для работы с временным хранилищем заметок.
2. В тестах использовать `setUp` для создания временной папки.
3. Использовать `tearDown` или `addCleanup` для удаления временных файлов.
4. Проверить добавление и чтение данных.
5. Проверить, что каждый тест работает независимо.

## Команда запуска

```bash
python -m unittest tests.test_lab_04_fixtures -v
```

## Вывод

В работе показана подготовка тестового окружения и очистка ресурсов после выполнения тестов.
