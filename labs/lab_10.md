# Лабораторная работа 10

## Тема

Мокирование файлов, времени и внешних сервисов.

## Цель работы

Показать, как тестировать код без реального обращения к внешним ресурсам.

## Задание

1. Создать функцию, которая зависит от текущего времени.
2. Создать функцию, которая работает с внешним клиентом.
3. В тестах заменить внешнюю зависимость через mock.
4. Проверить успешный и ошибочный сценарии.

## Команда запуска

```bash
python -m unittest tests.test_lab_10_external -v
```

## Вывод

В работе показано, как изолировать тесты от внешних зависимостей.
