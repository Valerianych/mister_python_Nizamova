# Лабораторная работа 3

## Тема

Запуск тестов и автоматический поиск тестов.

## Цель работы

Научиться запускать тесты через командную строку и использовать автоматический поиск тестов через `unittest discover`.

## Задание

1. Разместить тесты в отдельной папке `tests`.
2. Назвать файлы с тестами по шаблону `test_*.py`.
3. Запустить все тесты одной командой.
4. Проверить, что `unittest` сам находит тестовые файлы.
5. Добавить короткое описание команд запуска в README.

## Команды для запуска

Запуск всех тестов:

```bash
python -m unittest discover -s tests -v
```

Запуск отдельного файла:

```bash
python -m unittest tests.test_lab_03 -v
```

## Что должно быть в результате

После запуска команды должны выполниться все тесты из папки `tests`.

## Вывод

В ходе работы был изучен запуск тестов через командную строку и автоматический поиск тестов в проекте.
