# Лабораторная работа 11

## Тема

Проверка предупреждений и логов.

## Цель работы

Научиться проверять предупреждения через `assertWarns` и сообщения логов через `assertLogs`.

## Задание

1. Создать функцию, которая выдаёт предупреждение при устаревшем параметре.
2. Создать функцию, которая пишет сообщение в лог.
3. Написать тесты для предупреждения и логов.

## Команда запуска

```bash
python -m unittest tests.test_lab_11_logs_warnings -v
```

## Вывод

В работе показано тестирование предупреждений и логирования.
