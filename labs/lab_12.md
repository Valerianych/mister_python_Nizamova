# Лабораторная работа 12

## Тема

Асинхронные тесты в `unittest`.

## Цель работы

Научиться тестировать асинхронный код через `IsolatedAsyncioTestCase` и `AsyncMock`.

## Задание

1. Создать асинхронную функцию.
2. Создать сервис, который вызывает асинхронный клиент.
3. Написать тесты на `IsolatedAsyncioTestCase`.
4. Использовать `AsyncMock` для подмены асинхронной зависимости.

## Команда запуска

```bash
python -m unittest tests.test_lab_12_async -v
```

## Вывод

В работе показано тестирование асинхронных функций стандартным модулем `unittest`.
