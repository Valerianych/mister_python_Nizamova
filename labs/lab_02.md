# Лабораторная работа 2

## Тема

Основные проверки в `unittest`.

## Цель работы

Изучить основные методы проверки, которые используются в тестах.

## Задание

1. Добавить несколько функций для работы с числами и строками.
2. Написать тесты с использованием разных проверок.
3. Проверить обычные значения и ошибки.

## Какие проверки использовать

- `assertEqual` — сравнение ожидаемого и фактического значения;
- `assertTrue` — проверка истинности условия;
- `assertIn` — проверка наличия элемента;
- `assertListEqual` — сравнение списков;
- `assertRaises` — проверка возникновения ошибки.

## Что должно быть в результате

В проекте должны быть тесты, которые проверяют не только правильные значения, но и ошибочные ситуации.

## Пример запуска

```bash
python -m unittest tests.test_lab_02 -v
```
