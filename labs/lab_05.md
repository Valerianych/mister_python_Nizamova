# Лабораторная работа 5

## Тема

`TestSuite` и `TestLoader`.

## Цель работы

Научиться собирать набор тестов вручную и запускать только выбранные тесты.

## Задание

1. Создать отдельный файл с тестами.
2. Создать функцию `load_tests` или отдельный runner.
3. Сформировать `TestSuite` через `unittest.TestLoader`.
4. Запустить выбранные тесты отдельно от общего discovery.

## Команда запуска

```bash
python tests/run_lab_05_suite.py
```

## Вывод

В работе показано, как можно управлять набором тестов вручную.
