# Лабораторная работа 8

## Тема

`patch`, `patch.object`, `patch.dict`.

## Цель работы

Научиться временно подменять функции, методы и переменные окружения во время теста.

## Задание

1. Создать функцию, которая читает настройку из переменных окружения.
2. В тестах использовать `patch.dict` для подмены окружения.
3. Использовать `patch` для подмены функции.
4. Проверить, что после теста окружение не ломается.

## Команда запуска

```bash
python -m unittest tests.test_lab_08_patch -v
```

## Вывод

В работе показано временное изменение зависимостей через `patch`.
