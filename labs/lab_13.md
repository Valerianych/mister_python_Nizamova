# Лабораторная работа 13

## Тема

Поддерживаемость тестов, CI и автоматический запуск.

## Цель работы

Настроить автоматический запуск тестов и описать команды для проверки проекта.

## Задание

1. Проверить, что все тесты запускаются одной командой.
2. Добавить GitHub Actions workflow.
3. Указать версию Python.
4. Настроить установку проекта через `pip install -e .`.
5. Запустить тесты через `unittest discover`.

## Команда локального запуска

```bash
python -m unittest discover -s tests -t . -v
```

## Вывод

В работе показана базовая настройка CI для автоматической проверки тестов.
